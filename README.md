# PUNKIEZ Landing Page

## Setup Complete âœ…

- Next.js 15 with App Router & TypeScript
- Tailwind CSS v4 with custom orange brand color (#FF7A00)
- Glass morphism card design
- Mobile-first responsive layout
- Accessibility features (aria-labels)

## Development

```bash
pnpm dev      # Start dev server
pnpm build    # Build for production
pnpm start    # Start production server
```

## Add Your Logos

Drop these two images into `/public`:

1. **punkiez-logo.png** - Wordmark logo (PUNKIEZ text)
2. **p-icon.png** - P icon/stamp

The page is configured to display them automatically.
